# Calendar for sport events

**Calendar for sport events**  A simple web-based app in which you can check the upcoming sport events and add new ones. 

---

## Features

- Add Events 
- Save events locally using `localStorage`
- Responsive and simple UI


## Tech Stack

- HTML
- CSS
- JavaScript
- Browser `localStorage`

## How to Use

1. Open `calendar_view.html` in a browser.
2. Check the upcoming events in november 2025
3. Click **"Events"** to save check the details.
4. Go to **"Add Events"** to add new events which will be displayed on the calendar
5. Click on Calendar to go back.

> Note: Your data is stored in the browser using `localStorage`, so it stays saved even if you close the tab.
